# Twitter Sentiment Analysis
## Problem Statement
This project analyzes tweets to classify sentiment as positive or negative.
## Dataset
- Source: Kaggle
- Size: 1600000 tweets
## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- NLTK
## Models Used
- Logistic Regression
## Results
- Best Accuracy: 77.7%
## How to Run
1. Use Google Colab
2. Install required libraries
3. Train the model
