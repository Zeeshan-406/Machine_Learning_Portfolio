# Fake News Detection System
## Problem Statement
This project analyzes either the news is True or Fake.
## Dataset
- Source: Kaggle
- Size: 44898 News
## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Wordopt
- TfidfVectorizer
## Models Used with Result
- Logistic Regression 98.7%
- DecisionTreeClassifier 99.6%
- RandomForestClassifier 98.9%
- GradientBoostingClassifier 99.5%
## How to Run
1. Use Google Colab
2. Install required libraries
3. Train the model
